# ESpai de DAdes Turístiques de les Illes Balears
Web oficial: https://ibtourismdataspace.org/

L’ESDATIB és l’Espai de Dades Turístiques de les Illes Balears, una iniciativa orientada a centralitzar, visualitzar i facilitar l’accés a dades turístiques rellevants per a la comunitat, empreses i administracions.
